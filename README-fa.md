# DS2 CCM Font Editor

یک ویرایشگر گرافیکی مدرن برای فایل‌های فونت **.ccm** بازی **Dark Souls II** (و نسخه Scholar of the First Sin).

این ابزار به شما امکان می‌دهد فایل‌های CCM را باز کنید، گلیف‌ها را مشاهده و ویرایش کنید، گلیف اضافه یا حذف کنید، تکسچرهای مرتبط (PNG/DDS) را پیش‌نمایش دهید، گلیف‌ها را به فرمت متنی ساده اکسپورت/ایمپورت کنید و فایل CCM تغییر یافته را ذخیره کنید. ابزار ایده‌آلی برای مددرهایی که روی فونت‌های سفارشی، ترجمه‌ها یا تغییرات رابط کاربری کار می‌کنند.

## ویژگی‌ها
- باز کردن و ذخیره کامل فایل‌های CCM
- جدول قابل ویرایش با تمام پارامترهای گلیف (کد یونیکد، Texture ID، prespace، width، advance، مناطق تکسچر)
- اضافه و حذف گلیف
- اکسپورت/ایمپورت گلیف‌ها به/از فایل متنی ساده برای ویرایش آسان یا مقایسه تغییرات
- تشخیص خودکار و پیش‌نمایش تکسچرهای مرتبط:
  - `filename.png` یا `filename.dds` → تکسچر شماره ۰
  - `filename_1.png`، `filename_0001.png` و غیره → تکسچرهای بعدی
- رابط کاربری مدرن با تم تاریک (الهام‌گرفته از Dracula)
- لاگ‌گیری دقیق برای دیباگ (`ccm_editor_detailed.log`)
- ابزارهای دیباگ: نمایش اطلاعات هدر و دامپ کامل هگز فایل

## نصب

```bash
git clone https://github.com/ReaIMoFraD/DS2-CCM-Font-Reader.git
cd DS2-CCM-Font-Reader
pip install -r requirements.txt
python main.py
```


## نحوه استفاده
1. روی دکمه **Load CCM** کلیک کنید و یک فایل `.ccm` را انتخاب کنید (معمولاً در پوشه `menu/font` بازی قرار دارد).
2. ابزار به‌طور خودکار تکسچرهای مرتبط را در همان پوشه جستجو و نمایش می‌دهد.
3. مقادیر گلیف‌ها را مستقیماً در جدول ویرایش کنید یا از **Add Glyph** / **Delete Glyph** استفاده کنید.
4. برای ویرایش دسته‌جمعی یا بک‌آپ، از **Export to Text** استفاده کنید و سپس تغییرات را با **Import from Text** اعمال کنید.
5. تغییرات را با **Save CCM** ذخیره کنید.

## کارهای آینده / ویژگی‌های برنامه‌ریزی شده
- پیش‌نمایش مستقیم تکسچر با هایلایت منطقه گلیف
- پشتیبانی کامل از فرمت DDS با تولید thumbnail
- رندر پیش‌نمایش گلیف روی تکسچر
- ویرایش drag-and-drop منطقه تکسچر روی پیش‌نمایش
- پردازش دسته‌ای چندین فایل CCM
